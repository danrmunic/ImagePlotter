Detailed Description

A motor class implements a motor driver for an ME405 kit.

Objects of this class can be used to apply PWM to a given DC motor.

Set the PWM duty cycle for the motor channel.

This method sets the duty cycle to the motor to the given level. Positive values cause effort in one direction, negative values in the opposite direction.

Parameters
