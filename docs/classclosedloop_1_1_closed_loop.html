Detailed Description

Interface with closed loop controller.

Contains all methods that will be used in task_hardware to set the duty cycle based on closed loop control.

Constructs a closed loop controller.

Sets up a PID controler, a set point and the saturation limits for a given controller.

Parameters
Kpis the proportional gain in the PID controller.
setpointis the goal set point of the controller. If none is specified setpoint = 0.
Kiis the integral gain in the PID controller. If none is spesified Ki = 0.
Kdis the derivative gain in the PID controller. If none is spesified Kd = 0.
satLimis a list containing the upper and lower bounds of the saturation limit. If none is spesified satLim is between -100 and 100.

def closedloop.ClosedLoop.update (   self,

Constructs a closed loop controller and Facilitates recording time and postion to the motor.

Uses the set point and the input parameter read to create an error signal. This error signal is converted into an output signal with PID controller. While recording is true a loop is used to record the position and time. Once it has finished recording, it prints the values of all 100 positons and times.

Parameters
readis the motor reading the new values from the encoder.
tdifis the difference of time between recordings.
Returns
Sends back actuation signal value using sat method.
